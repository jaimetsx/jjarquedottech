# jjarque.tech — Personal Portfolio

> My personal portfolio website, designed to showcase my projects, skills, and experience as a full-stack developer.

Live at: [jjarque.tech](https://jjarque.tech)

---

## Tech Stack

- [Next.js](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vercel Analytics](https://vercel.com/analytics)
- [Vercel Speed Insights](https://vercel.com/docs/speed-insights)
- [pnpm](https://pnpm.io/)
- [ESLint](https://eslint.org/)


---

## Getting Started

```bash
pnpm install
pnpm dev
pnpm build
pnpm start
```
